# 🔊 Audio Amplifier Using Discrete Components

## Overview
This project presents the design, simulation, and hardware implementation of a transistor-based audio amplifier using discrete electronic components. The amplifier was first designed and analyzed in **LTspice** and then physically implemented on a **breadboard** using transistors, resistors, and capacitors.

The project demonstrates the fundamental principles of analog electronics, including transistor biasing, signal amplification, and capacitor coupling techniques.

---

## Objectives

- Design an audio amplifier using discrete components.
- Simulate the circuit using LTspice.
- Implement the circuit on a breadboard.
- Verify the amplification of low-amplitude audio signals.
- Compare simulation and practical results.

---

## Circuit Operation

1. The input audio signal is applied through a coupling capacitor.
2. The transistor stage amplifies the weak input signal.
3. Capacitors provide AC coupling and stabilize the circuit operation.
4. The amplified signal is obtained at the output and can drive a small speaker or headphones.

## Simulation

The amplifier circuit was simulated in LTspice to verify:

- Voltage gain
- Output waveform
- Frequency response
- Proper transistor biasing


## Hardware Implementation

The simulated circuit was implemented on a breadboard using discrete components.
